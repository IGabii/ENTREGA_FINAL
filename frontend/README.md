PhotoRental: Plataforma de Alquiler de Equipos (Resumen Ejecutivo)

¿Qué es y Para qué sirve?

PhotoRental es una solución completa de comercio electrónico construida con una arquitectura de separación clara de responsabilidades (Front-end y Back-end operan de manera independiente). La plataforma digitaliza y optimiza la gestión de inventario y el proceso de alquiler de equipos fotográficos profesionales, sirviendo como un portal de consulta para clientes y una herramienta de administración (CRUD) para el staff interno.

Tecnologías Aplicadas (El Stack)

Front-end: Nos apoyamos en React.js para entregar una SPA (Single Page Application) rápida y sin interrupciones, garantizando una experiencia de navegación instantánea. El diseño, totalmente responsive, se logró con Tailwind CSS, lo que asegura una visualización perfecta y profesional en cualquier dispositivo (móvil o desktop).

Back-end: La plataforma corre sobre Python/Django, que expone una API RESTful robusta a través de Django REST Framework, fundamental para gestionar el inventario y la lógica de negocio.

Seguridad: Para blindar el acceso a la administración, utilizamos el protocolo de seguridad JWT (JSON Web Tokens), que gestiona sesiones efímeras mediante tokens cifrados.
